WebGL Garbage

A demo of webgl. WebGL takes the web to the next level. One can now use their graphics card through the web browser. This allows web applications to leverage decent computational power. I'm personally excited by the fact that I can include some javascript on my static webpage that could for example, simulate Maxwells Equations (the equations that describe how light behaves under the wave model of light). This means that for example, I could write my simulation in javascript and then present and share my simulation realtime during a meeting.

I'm going to be practicing WebGL, which means I need to actually do WebGL. Thus I've made a separate webpage WebGL Garbage, which has my exercies thus far.
